GPU Gems

Author: Nvidia

Category: Non‑fiction

Published: 2004

Status: Unavailable

Description

GPU Gems is a comprehensive collection of practical techniques, tips and tricks for real‑time graphics programming using modern GPUs. The book gathers chapters from leading graphics developers and researchers, covering natural effects, lighting and shadows, materials, image processing, performance optimization and beyond.

Designed for intermediate to advanced readers working in game development, real‑time rendering, or graphics research, GPU Gems combines full‑color diagrams, screenshots, code samples and demos to illustrate complex graphics algorithms and shader techniques.
